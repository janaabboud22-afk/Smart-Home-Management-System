# Smart Home Management System

A Java-based smart home management system designed to monitor and control household devices through a centralized platform.

## Features

- Device management
- Room organization
- Appliance monitoring
- Smart lighting control
- Centralized home management

## Technologies

- Java
- Object-Oriented Programming (OOP)
- Software Design

## Project Structure

- Device.java
- Appliance.java
- Light.java
- Room.java
- ManagementSystem.java
- run.java

## Skills Demonstrated

- Object-Oriented Programming
- System Design
- Problem Solving
- Software Development
